- **إدارة الأذونات**: عرض وشرح الأذونات الحساسة باللغة العربية.
- **كشف التتبع**: التعرف على SDKs التتبع الموجودة داخل التطبيقات (مثل Facebook, Firebase, AdMob).
- **تحليل النطاقات**: استخراج الروابط والنطاقات (Domains) التي يتصل بها التطبيق.
- **تقييم الأمان**: إعطاء درجة أمان لكل تطبيق بناءً على سلوكه وأذوناته.
- **نظام Pro**: ميزات متقدمة للتحليل العميق.

# Privacy Guardian AR

An advanced Android application to inspect the privacy and security of installed apps. Works 100% offline to ensure maximum privacy.

## Key Features
- **App Scanner**: Comprehensive analysis of all installed applications.
- **Permission Manager**: View and explain sensitive permissions.
- **Tracker Detection**: Identify tracking SDKs (e.g., Facebook, Firebase, AdMob).
- **Domain Analysis**: Extract endpoints and domains within the APK.
- **Security Score**: Provide a security rating for each app.
- **Pro Mode**: Advanced features for deep analysis.
